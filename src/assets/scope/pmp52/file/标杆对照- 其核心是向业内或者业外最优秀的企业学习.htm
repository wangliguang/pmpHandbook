    PMBOK工具和技术详解-标杆对照

    PMP-王干一
    2019-04-29 14:58:55
    15篇 作品
    1245 总阅读量

    在PMBOK中有5.2，8.1，13.2三个子过程用到了标杆对照。

    标杆对照是将本项目的相关指标和可比项目的实践进行比较，以便识别最佳实践，形成改进意见，并为绩效考核提供依据。作为标杆的项目可以来自执行组织内部或外部，或者来自同一应用领域或其他应用领域。标杆对照也允许用不同应用领域或行业的项目做类比。

    起源

    上世纪70年代由施乐公司首开标杆对照先河。

    目的

    不断寻找和同行一流公司的最佳实践，并以为基准和本企业进行比较、分析、判断，从而使自己企业得到不断改进，进入或赶超一流公司，创造优秀业绩的良性循环过程。

    其核心是向业内或者业外最优秀的企业学习。通过学习企业重新思考和改进经营实践，创造自己的最佳实践，这实际上是模仿创新的过程。

    标杆对照步骤


    工作和生活中的应用

    竞品分析就是一种常见的标杆对照方法。

    常说的“你看别人的老公”，“别人家的孩子”，这也是一种标杆对照方法。当然提醒别盲目比较哦，毕竟每个人都有自己的特点！
